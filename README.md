# GlobalTrade Landing Page

This repository contains the landing page for **GlobalTrade**, a modern platform focused on global commerce. The project is built using **Vite** and **React/JavaScript**, emphasizing a fully responsive user interface.

---

## 🚀 Technologies Used

* **Vite**: Next-generation frontend build tool.
* **Node.js**: JavaScript runtime environment (Version 20.19+ or 22.12+).
* **React / JavaScript (ES6+)**: Core logic and UI components.
* **CSS / Frameworks**: Responsive design and mobile-first approach.

---

## 🛠️ Prerequisites

To run this project locally, ensure you have the following installed:

* **Node.js**: Version **20.19.0** or higher (LTS recommended). 
    > *Note: Older versions like 16.x will cause compatibility errors with Vite.*
* **npm**: Official Node.js package manager.

---

## 💻 Installation & Local Execution

Follow these steps to set up the environment on your machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/gabrielamachin/globaltrade-landing.git](https://github.com/gabrielamachin/globaltrade-landing.git)
    cd globaltrade-landing
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Access the application:**
    Once the server is running, open your browser at `http://localhost:5173` (or the port indicated in your terminal).
